isShortHex

  • isShortHex(val: string, prefix?: string): boolean
    • Defined in utils.ts:98

    Determines if a hex string is in CSS shorthand notation.

    example
    isShortHex('#000') // Returns `true`
isShortHex('0x000', '0x') // Returns `true`
isShortHex('0x000000', '0x') // Returns `false`
isShortHex('#000', '0x') // Returns `false`

    Parameters

    • val: string

      The hex string to test against.

    • Default value prefix: string = "#"

      Prefix to look out for.

    Returns boolean

    true if the hex string is in shorthand notation, false otherwise.

parseUnit

  • parseUnit(val: string | number, defaultUnit?: string): [number, string]
    • Defined in utils.ts:148

    Parses a number or string representing a number with unit and returns a tuple with the number value and string unit. If a number is provided, the default unit is px.

    example
    parseUnit('1px') // Returns [1, 'px']
parseUnit('1em', 'em') // Returns [1, 'em']

    Parameters

    • val: string | number
    • Default value defaultUnit: string = "px"

      Default unit if the unit is absent.

    Returns [number, string]

    Parsed value-unit tuple.
