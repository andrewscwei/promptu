font

  • font(family: string, size?: string | number, weight?: FontWeight, style?: FontStyle, lineHeight?: LineHeight, letterSpacing?: LetterSpacing, variant?: FontVariant): string
    • Defined in typography.ts:208

    CSS mixin for defining font style related rules.

    example
    font('Roboto', '16px') // Returns...
    {
  font-family: 'Roboto';
  font-size: 16px,
  font-style: normal,
  font-weight: 400,
  font-variant: normal;
  line-height: normal;
  letter-spacing: normal;
}

    Parameters

    • family: string

      Font family.

    • Default value size: string | number = "1.6rem"

      Font size. If the value specified is a number, the associated unit will be 'rem'.

    • Default value weight: FontWeight = fontWeights.normal

      Font weight.

    • Default value style: FontStyle = "normal"

      Font style.

    • Default value lineHeight: LineHeight = "normal"

      Line height. If the value specified is a number, the associated unit will be 'rem'.

    • Default value letterSpacing: LetterSpacing = "normal"

      Letter spacing. If the value specified is a number, the associated unit will be 'rem'.

    • Default value variant: FontVariant = "normal"

      Font variant.

    Returns string

    Generated CSS rules.

fontFace

  • fontFace(family: string, src: string, weight?: FontWeight, style?: FontStyle, display?: FontDisplay): string
    • Defined in typography.ts:163

    CSS mixin for defining @font-face rules.

    example
    fontFace('Roboto', require('fonts/Roboto.ttf')) // Returns...
    [at]font-face {
  font-family: 'Roboto';
  src: url('fonts/Roboto.ttf') format('truetype');
  font-style: normal;
  font-weight: 400;
  font-display: auto;
}

    Parameters

    • family: string

      Font family.

    • src: string

      File path.

    • Optional weight: FontWeight

      Font weight, automatically inferred if unspecified.

    • Optional style: FontStyle

      Font style, automatically inferred if unspecified.

    • Optional display: FontDisplay

      Font display, automatically inferred if unspecified.

    Returns string

    Generated CSS rules.
